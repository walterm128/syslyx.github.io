ASISTENCIA REMOTA

ULTRAVIEWER (RECOMENDADO PC)


1. Vamos al siguiente link: https://ultraviewer.net/en/download.html
2. Elegimos Download o bajar


3. Ejecuta el archivo de instalación descargado
4. Selecciona Next > 3 veces
5. Por último Install

6. Después ejecutar el icono del escritorio:

7. Enviar foto o datos ID y Contraseña


8. Consultar por transferencia bancaria (CBU)

ANYDESK (RECOMENDADO PC Y ANDROID)


1. Vamos al siguiente link para Windows:  https://anydesk.com/es
2. O para android vamos al siguiente link: Android
3. Elegimos Download o bajar

4. Al final si pregunta de instalarlo le damos instalar


5. Consultar por transferencia bancaria (CBU)


TEAMVIEWER


1. Entra en https://www.teamviewer.com/es
2. Haz clic en el botón Descargar TeamViewer
3. Sigue las instrucciones para guardar el archivo de instalación


4. Ejecuta el archivo de instalación descargado
5.Selecciona Instalar


6. Después ejecutar el icono del escritorio


7. Enviar foto o datos ID y Contraseña


8. Consultar por transferencia bancaria (CBU)
